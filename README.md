# Проект: Samurai Social Network
----------------------------------

Социальная сеть для веб разработчиков, где можно общаться в общем чате, подписываться друг на друга, создавать и редактировать свой логин 

**Используемые технологии:**
1. TypeScript
2. Grid Layout
3. React
4. Redux/Redux-thunk
